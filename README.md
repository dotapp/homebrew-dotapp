# homebrew-dotapp

[dotapp][] casks for [Homebrew Cask][].

[dotapp]: https://dotapp.github.io/
[homebrew cask]: http://caskroom.io

## Installation

One-time only:

```shell
brew tap dotapp/dotapp
```

Then install a Cask, e.g.:

```shell
brew cask install manganese
```
